# 管道微小泄漏智能诊断 Agent (POC)

## 项目简介
本项目是一个概念验证（Proof of Concept）模型，旨在探索大语言模型（LLM）在多分支管网高频时序信号分析与故障诊断中的自动化落地。
传统的管网微小泄漏检测依赖人工排查和固定的阈值判定，面对 2048Hz 采样率产生的大量高频 TDMS/LVM 日志数据时，排障效率极低且容易漏判瞬态微弱信号。

本项目构建了一个自动化的诊断流水线，将信号的频域特征（如 HHT 边际谱特征、能量突变）转化为结构化文本，驱动 AI Agent 进行长链推理，最终输出专业的工程排障建议。

## 核心工作流
1. **数据截取**：自动读取并定位管网传感器日志中的异常波动区间。
2. **特征结构化**：提取异常频段特征，构建 Agent 诊断所需的上下文。
3. **推理与诊断**：调用大模型 API 研判是“机械故障”还是“微小泄漏”，并生成排障建议。

## 技术栈
* Python 3.x
* 信号处理 (Simulated HHT / Wavelet analysis)
* 大模型 API (OpenAI SDK 格式)
